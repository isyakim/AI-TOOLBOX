# AI工具箱 (AI Toolbox)

一个功能强大的桌面AI助手应用，基于Electron构建，提供智能对话、AI工具集、多会话管理等丰富功能。

## ✨ 主要特性

### 🤖 智能对话
- **流式响应**: AI回复实时显示，提供更流畅的交互体验
- **多会话管理**: 支持创建、切换和管理多个独立对话会话
- **消息操作**: 复制、删除消息，支持快捷操作
- **历史记录**: 自动保存对话历史，支持会话恢复
- **Markdown渲染**: 支持代码高亮、格式化显示

### 🛠️ AI工具集
提供6个专业AI工具：
- **🌐 文本翻译**: 支持多种语言互译（中文、英语、日语、韩语等）
- **📝 文本摘要**: 快速生成不同长度的文章摘要
- **💻 代码助手**: 代码生成、解释、优化和调试
- **✍️ 写作助手**: 文章、邮件、报告等内容创作
- **✅ 语法检查**: 检查并纠正语法错误
- **💡 概念解释**: 解释复杂概念和术语

### 🔐 数据安全
- **加密存储**: 使用AES-GCM加密算法保护敏感数据
- **密码保护**: 支持设置密码保护API密钥等敏感信息
- **本地存储**: 所有数据仅存储在本地，不上传服务器

### 🎨 主题切换
内置5个精美主题：
- 🌞 浅色主题
- 🌙 深色主题
- 🌊 海洋蓝主题
- 💜 紫罗兰主题
- 🌿 森林绿主题

### 🔄 自动更新
- 定期检查更新（24小时间隔）
- 发现新版本时显示通知
- 支持手动检查更新
- 一键下载最新版本

### ⚙️ 多服务商支持
- OpenAI
- DeepSeek
- Azure OpenAI
- 自定义API服务

## 📖 使用指南

### 1. 配置API
首次使用需要配置AI服务：
1. 点击左侧菜单的"API配置"
2. 选择服务商（OpenAI、DeepSeek等）
3. 输入API密钥
4. 选择默认模型
5. 点击"测试连接"验证配置
6. 点击"保存配置"

### 2. 开始对话
1. 点击"智能对话"进入聊天界面
2. 在左侧可以创建新对话或切换已有会话
3. 在输入框输入问题，按Enter或点击"发送"
4. AI会实时流式返回回复
5. 鼠标悬停在消息上可以复制或删除

### 3. 使用AI工具
1. 点击"AI工具集"
2. 选择需要的工具（翻译、摘要、代码助手等）
3. 填写相应参数
4. 点击执行按钮
5. 查看AI生成的结果

### 4. 管理密钥
1. 点击"密钥管理"查看所有保存的API配置
2. 可以切换使用不同的配置
3. 删除不需要的配置

### 5. 切换主题
- 点击右上角的🎨图标
- 主题会循环切换
- 设置会自动保存

## 🔑 快捷键

- `Enter`: 发送消息
- `Shift + Enter`: 换行

## 🛠️ 技术栈

- **Electron**: 跨平台桌面应用框架
- **JavaScript ES6+**: 现代JavaScript
- **CSS3**: 响应式UI设计
- **Web Crypto API**: 数据加密
- **LocalStorage**: 本地数据存储

## 🔒 隐私与安全

- 所有数据仅存储在本地计算机
- API密钥可选加密存储
- 不收集任何用户数据
- 不上传任何信息到第三方服务器
- 开源透明，代码可审查

# AI工具箱 (AI Toolbox)

一个功能强大的桌面AI助手应用，基于Electron构建，提供智能对话、AI工具集、多会话管理、主题切换及多服务商统一接入等能力。

## ✨ 主要特性

### 🤖 智能对话
- **流式/中止控制**: 回复实时显示，可在生成过程中随时点击“停止”
- **多会话管理**: 左侧栏展示会话列表，支持创建、切换、重命名及删除
- **Roo 行为角色**: 预置多种回答风格（默认助手/代码专家/产品参谋/思辨导师），会作为系统提示影响 AI 输出
- **动态模型下拉**: 下拉列表自动读取最新测试通过的模型集合
- **Markdown渲染**: 支持代码高亮、行内代码、列表等
- **消息操作**: 复制、删除、快速定位，自动滚动到底部

### 🛠️ AI工具集
提供6个专业AI工具：
- **🌐 文本翻译**: 支持多种语言互译（中文、英语、日语、韩语等）
- **📝 文本摘要**: 快速生成不同长度的文章摘要
- **💻 代码助手**: 代码生成、解释、优化和调试
- **✍️ 写作助手**: 文章、邮件、报告等内容创作
- **✅ 语法检查**: 检查并纠正语法错误
- **💡 概念解释**: 解释复杂概念和术语

### 🔐 数据安全
- **加密存储**: 使用AES-GCM加密算法保护敏感数据
- **密码保护**: 支持设置密码保护API密钥等敏感信息
- **本地存储**: 所有数据仅存储在本地，不上传服务器

### 🎨 主题切换
内置5个精美主题：
- 🌞 浅色主题
- 🌙 深色主题
- 🌊 海洋蓝主题
- 💜 紫罗兰主题
- 🌿 森林绿主题

### 🔄 自动更新
- 定期检查更新（24小时间隔）
- 发现新版本时显示通知
- 支持手动检查更新
- 一键下载最新版本

### ⚙️ 多服务商支持
- 左侧“模型平台”列表内置多家服务商（OpenAI、OpenAI Compatible、Amazon Bedrock、Anthropic、DeepSeek、OpenRouter 等），可搜索、切换、重命名自定义服务
- **动态模型管理**: 每个服务商可多选启用模型并支持手动添加
- **API 自动检测**: 进入应用即检测密钥管理里所有配置，若存在可用即点亮顶部状态；也可点击刷新按钮手动重测
- **统一测试面板**: “测试连接”会依次校验所有勾选模型，并输出详细结果列表

## 📖 使用指南

### 1. 配置API
1. 打开“设置 → API配置”，在左侧选择服务商（或自定义）
2. 输入 API Key / Base URL，可查看官方文档链接
3. 在“模型管理”中搜索、勾选或手动添加模型
4. 点击“测试连接”进行逐模型验证，测试成功后顶部状态会显示“已就绪”
5. 保存配置，可在“密钥管理”中快速切换/编辑/删除

### 2. 开始对话
1. 点击"智能对话"进入聊天界面
2. 在左侧可以创建新对话或切换已有会话
3. 在输入框输入问题，按Enter或点击"发送"
4. AI会实时流式返回回复
5. 鼠标悬停在消息上可以复制或删除

### 3. 使用AI工具
1. 点击"AI工具集"
2. 选择需要的工具（翻译、摘要、代码助手等）
3. 填写相应参数
4. 点击执行按钮
5. 查看AI生成的结果

### 4. 管理密钥
1. 点击"密钥管理"查看所有保存的API配置
2. 可以切换使用不同的配置
3. 删除不需要的配置

### 5. 切换主题
- 在侧栏底部点击“🎨 主题配色”，“弹出面板中直接选择主题
- 支持浅色、深色、海洋蓝、紫罗兰、森林绿等
- 聊天输入框等会随主题同步调整背景和文字色

## 🛠️ 技术栈
